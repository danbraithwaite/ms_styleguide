Colour Palette

These are the main colour palettes used accross M&S.com. Grey's are to be used primarily for text, line dividers and borders. Primary colours are to be used for brand purposes and highlights (e.g. sales & special offers). Function colours are to only be used to provide feedback to the user.

Grey Palette

Always use either #000000 or #333333 for text - all other greys will fail accessibility tests. #cccccc is commonly used for borders or 1px line dividers.

 Black #000000 Dark grey #333333 Medium Grey #999999 Light Grey #CCCCCC Background Grey #F8F8F8

Primary Palette

M&S Red to only be used for special offers, sales and promotions.

 Green #bdd755 Red (Sales/Offers) #c4201d

Function Palette

Success Green is the BG colour used in alert boxes. Error red is the colour for all alert copy and the border of all alert boxes. Error Pink is the BG colour used in alert boxes. Warning yellow is the colour for warning icons and borders of warning boxes. Warning yellow is the BG colour used in warning boxes.

 Success Green (BG Only) #f5f9e5 Error Red #EA122A Error Pink (BG Only) #fff3f4 Warning Yellow #ffcd00 Warning Yellow (BG Only) #ffffca Primary CTA Hover Color #a9c14b

Typography

When possible, use 6px baseline grid to align text elements. The majority of copy on M&S.com is M&S London. In situations where a more emotion heading is require, we use M&S Leeds. M&S Leeds is only to be used for top-level headings (H1 or H2).
