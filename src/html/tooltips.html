Tooltips

Tool tips should only be used when absolutely neccessary. The two current forms of tool tip are: 1) standard size with copy content, 2) Bag related actions - e.g. new item added. Bag related tooltips are generally more complex, include icons, a thumbnail and 2 call to actions.

Standard - Above

Text With Tooltip Tooltip above the content.

Tooltip height and width changes depending on copy. Don’t be stupid and make it massive, they aren’t designed for massive paragraphs...

  • Tooltip box has 5px rounded corners.
  • Fill colour: #f8f8f8.
  • Light drop shadow.
  • 20px padding around copy.
  • Font: 15px (Body), #333333

Standard - Below

Text With Tooltip Tooltip below the content.

When to use above, below, left and right tooltips depends on page layout and how much space you have - use your own judgement.

  • Tooltip box has 5px rounded corners.
  • Fill colour: #f8f8f8.
  • Light drop shadow.
  • 20px padding around copy.
  • Font: 15px (Body), #333333

Bag Tooltip

This style should only be used for the Bag section. When using icons, thumbnails and CTA’s inside the tooltip, make sure there is enough spacing so that it doesn’t look cramped.

  • Tooltip box has 5px rounded corners.
  • Fill colour: #333333
  • Light drop shadow.
  • 20px padding around content.
  • Font: 15px (Body)
  • Primary CTA & Link CTA
