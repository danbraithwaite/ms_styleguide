Buttons

Anything requiring user action should be styled as a button when possible. Try to limit the use of text-based CTA links.

Primary

Normal state
  • Font weight: medium
  • Font size: 15px
  • Colour: #333333
  • Button colour: #bdd755
 Hover state
  • Button colour: #bdd755
 Active/Pressed state
  • Button colour: #9db346
 Disabled state
  • Font colour: #cccccc
  • Button colour: #eeeeee
  • Use no-drop cursor
