Text Links

Text links/cta’s are always secondary to the main M&S calls to action. They should only be used when it isn’t practical to have a full CTA. Text links should be limited to a few words only, don’t go crazy and use a full sentence...Text colours can vary, usually they are #333333, but this can change depending on context. For now, text links are underlined, with the only hover interaction being the change in cursor - down the road this will potentially change, and a hover colour may be introduced.
