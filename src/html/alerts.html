Alerts

Feedback for users for success messages, alerts/information and errors. Alerts can be either with an icon or without. You can also use medium font weight for the important part of the message to help make an impact. All copy is 15px Medium/Regular. Try to keep it to one line...

Success

Example use case: “More in your bag than you expected? We saved the items you added to your bag on your last visit.”

Congratulations! This is what a success message looks like with one line of copy and an icon.
  • Font weight: medium
  • Font size: 15px
  • WColour: #333333
  • Icon colour: #bdd755
  • border: #bdd755
  • Box fill/bg: #f5f9e5

Congratulations! This is what a success message looks like with one line of copy and no icon.

Congratulations! This is what a success message looks like with two lines of copy and an icon:

  • We’ve gone ahead and applied a discount to your order!

Important info

Example use case: “Please note that all of our Christmas delivery order deadlines have now passed so we are no longer able to offer delivery before Christmas.”

Something important is happening and we want to make sure you are in the loop.
  • Font weight: medium
  • Font size: 15px
  • Colour: #333333
  • Icon colour: #ffcd00
  • border: #ffcd00
  • Box fill/bg: #ffffca

Error alert

Example use case: “We’re sorry - Striped Rope Tie Dress with StayNEW™ has been removed from your shopping bag”

We’re sorry but we’ve completely run out of stock since you added this to your bag.
  • Font weight: medium
  • Font size: 15px
  • Colour: #333333
  • Icon colour: #ea122a
  • Box border: #ea122a
  • Box fill/bg: #fff3f4
